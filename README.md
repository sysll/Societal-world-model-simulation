# 人群抑制性

本项目将疏散人群划分为小孩、老人和成年人三类，以刻画不同年龄群体在身体尺度、占用空间、行走能力以及灾害响应行为上的差异。个体生成时，系统按照设定比例随机分配人群类型，并为其赋予相应的身体尺寸和基础运动参数。其中，小孩占 20%，身高为 1.20 m，半径为 0.20 m，期望速度为 0.90 m/s；老人占 30%，身高为 1.60 m，半径为 0.28 m，期望速度为 0.95 m/s；成年人占 50%，身高为 1.75 m，半径为 0.32 m，期望速度为 1.35 m/s。

在上述参数中，身高用于描述个体的基本物理尺度，并为后续可视化、碰撞体设置以及特殊通行能力扩展提供依据；半径表示个体在水平面上的占用范围，主要用于人与人之间的硬碰撞分离、社会力排斥计算，以及随机生成起点时的最小间距约束；期望速度表示个体在无拥堵、无障碍条件下倾向达到的基础行走速度，是社会力驱动项和运动更新过程中的重要输入。

在火灾场景中，不同类型人员还被赋予差异化的紧急加速能力。当个体进入火灾警戒范围后，其期望速度会根据类型进行放大：成年人最高速度倍率为 1.50，小孩为 1.40，老人为 1.25，用于反映不同年龄群体在危险刺激下的逃离能力差异。同时，当烟雾浓度较高时，所有类型个体的速度统一降低至 0.80 倍，以模拟浓烟对视线、呼吸和行动能力造成的抑制影响。

此外，模型引入家庭关系权重，用于调节亲属之间的凝聚强度。小孩与成年人之间的关系权重最高，为 1.25，表示儿童在疏散过程中对成年人具有更强依赖；老人和成年人之间的权重为 0.90，体现成年人对老人的照顾倾向；老人和小孩之间的权重为 0.60，而同类成员之间的权重相对较低，以避免家庭成员之间产生过强吸引导致局部拥挤。通过上述设置，模型能够同时刻画个体异质性、家庭协同行为以及灾害环境下的人群速度响应。
