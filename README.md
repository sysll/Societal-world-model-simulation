# 人群层面特征

本项目将疏散人群划分为小孩、老人和成年人三类，以刻画不同年龄群体在身体尺度、占用空间、行走能力以及灾害响应行为上的差异。个体生成时，系统按照设定比例随机分配人群类型，并为其赋予相应的身体尺寸和基础运动参数。其中，小孩占 20%，身高为 1.20 m，半径为 0.20 m，期望速度为 0.90 m/s；老人占 30%，身高为 1.60 m，半径为 0.28 m，期望速度为 0.95 m/s；成年人占 50%，身高为 1.75 m，半径为 0.32 m，期望速度为 1.35 m/s。

在上述参数中，身高用于描述个体的基本物理尺度，并为后续可视化、碰撞体设置以及特殊通行能力扩展提供依据；半径表示个体在水平面上的占用范围，主要用于人与人之间的硬碰撞分离、社会力排斥计算，以及随机生成起点时的最小间距约束；期望速度表示个体在无拥堵、无障碍条件下倾向达到的基础行走速度，是社会力驱动项和运动更新过程中的重要输入。

在火灾场景中，不同类型人员还被赋予差异化的紧急加速能力。当个体进入火灾警戒范围后，其期望速度会根据类型进行放大：成年人最高速度倍率为 1.50，小孩为 1.40，老人为 1.25，用于反映不同年龄群体在危险刺激下的逃离能力差异。同时，当烟雾浓度较高时，所有类型个体的速度统一降低至 0.80 倍，以模拟浓烟对视线、呼吸和行动能力造成的抑制影响。

此外，模型引入家庭关系权重，用于调节亲属之间的凝聚强度。小孩与成年人之间的关系权重最高，为 1.25，表示儿童在疏散过程中对成年人具有更强依赖；老人和成年人之间的权重为 0.90，体现成年人对老人的照顾倾向；老人和小孩之间的权重为 0.60，而同类成员之间的权重相对较低，以避免家庭成员之间产生过强吸引导致局部拥挤。通过该设置，模型能够刻画家庭成员在疏散过程中的协同行为。

在运动更新过程中，个体还受到路径引导力的作用。路径引导力由 Detour 全局路径规划结果提供方向基础，个体不会简单地沿路径中心线运动，而是在路径前方选取一定距离的局部目标点，并结合横向路径带偏移形成个体化的行进方向。该机制能够在保持总体朝向出口或安全目标运动的同时，避免所有人员集中在同一条 Detour 中心线上排队前进。若个体偏离路径过远，系统会施加较弱的路径修正，使其逐步回到可行路径带内，而不是强制投影到最近路径点，从而保留一定的局部分散性和运动自然性。

为进一步避免 Detour 规划后大量人员沿同一条直线路径行走，模型引入路径带分散机制。具体而言，Detour 仍负责生成全局可行路径，但个体在跟随路径时并不直接追踪当前路径点，而是沿路径方向向前选取约 2.4 m 的前视目标，并根据个体编号分配稳定的横向偏移量。横向偏移范围设置在约 $\pm 1.2,\mathrm{m}$ 内，使不同个体在同一条全局路径附近形成若干并行行进带，而不是完全重合在路径中心线上。该偏移在整个运动过程中保持相对稳定，因此能够减少人群排成单列的现象，使疏散流线更加接近真实人群在走廊或开阔区域中的分散行走行为。

同时，路径带机制采用较弱的路径拉回策略。只有当个体明显偏离全局路径带时，系统才会施加修正力，将其逐步引导回更接近目标方向的路径区域；在正常情况下，个体可以在路径带内保持一定横向自由度。相比每帧强制投影到 Detour 路径中心线的方法，这种方式能够避免局部避让和社会力造成的分散效果被抹除，从而提高人群运动的自然性和稳定性。

综合来看，本模型通过异质人群参数、火灾速度响应、家庭关系凝聚力、路径引导力以及路径带分散机制共同作用，使个体既能遵循全局疏散方向，又能表现出年龄差异、亲属协同、局部避让和非单列化行走行为，从而提高疏散模拟的真实性与可解释性。



# 场景特征
## 暴雨的疏散场景
在暴雨疏散场景中，环境不再被视为均质可通行空间，而是根据遮蔽条件和暴露程度被划分为不同的通行区域。系统在 Recast 构建 NavMesh 的过程中，对每个可行走区域进行暴雨暴露识别。该过程使 NavMesh 不仅表达空间可通行性，还能够进一步表达灾害环境属性。

在路径规划层面，暴雨区域会被赋予更高的通行代价。遮蔽区域保持默认代价，表示人员可以正常通行；露天暴雨区域的代价显著提高，表示行人在疏散过程中倾向于优先选择更加干燥、安全或舒适的路径。当路径必须穿过露天区域时，Detour 仍然允许通行，但会在可行路径中尽量降低暴雨暴露时间。因此，暴雨场景下的疏散路径并不单纯追求几何距离最短，而是在路径长度、通行安全性和暴露程度之间进行综合权衡。

在行为响应层面，暴雨会改变个体的运动速度。当前模型中，处于暴雨暴露区域的行人速度会提高，用于模拟人在强降雨环境下希望尽快离开露天区域、进入遮蔽空间或安全区域的行为倾向。该速度变化并不直接替代路径规划，而是作为局部运动更新中的速度修正项，与全局路径方向、社会力避让和碰撞约束共同作用。

此外，暴雨场景还考虑了靠近墙体或障碍物边界的通行代价。距离墙体较近的区域会被赋予更高代价，使路径规划尽量避开贴边行走，从而减少人员擦碰障碍物或在狭窄边界区域拥堵的情况。暴雨暴露代价与近墙代价可以叠加，使系统能够同时表达“避雨”和“保持安全通行间距”两类需求。

综上，暴雨场景的主要特征体现在三个方面：第一，NavMesh 在构建阶段即标记遮蔽区域与露天暴雨区域，使环境灾害属性直接参与路径规划；第二，Detour 根据不同区域的动态通行代价，引导人员优先选择遮蔽路径并减少暴雨暴露；第三，个体在暴雨区域产生速度响应，尽快脱离不利环境。通过上述设置，模型能够模拟暴雨条件下行人避雨、绕行、加速穿越露天区域以及保持安全通行距离等行为特征。


## 火灾的疏散场景
在火灾疏散场景中，环境不再是静态安全空间，而是随时间持续变化的动态危险场。系统首先在 NavMesh 上选取若干起火点，并将其映射到对应的可行走多边形上。火灾不会以简单圆形范围扩散，而是沿 NavMesh 多边形之间的邻接关系逐步传播，因此其蔓延过程受到空间连通性、楼层高差、通道结构和障碍物分布的共同影响，能够避免火焰穿墙扩散等不合理现象。

火灾传播过程中，每个 NavMesh 多边形都会维护火焰强度、烟雾浓度、点燃进度、危险等级和通行状态等动态属性。相邻多边形之间根据火源强度、燃料系数、距离和坡度计算点燃进度，当点燃进度达到阈值后，该区域被判定为起火区域。火焰强度随时间先增长后衰减，烟雾则会从燃烧区域向相邻区域扩散，并在向高处传播时具有更强扩散趋势。

在路径规划层面，火灾状态会动态改变 NavMesh 多边形的通行代价。轻度烟雾区域会提高通行代价，使人员倾向于绕开；火焰或高烟雾区域会进一步提高代价，表示该区域具有较高危险性；当火焰强度超过阻断阈值后，对应多边形会被标记为不可通行区域，Detour 在后续路径规划中将无法穿越该区域。由此，人员路径会随着火势发展而动态调整，而不是始终沿初始最短路径运动。

在人员行为层面，火灾会引入速度响应机制。当个体位于火灾警戒范围内时，其期望速度会根据年龄类型进行提高，用于模拟人员在危险刺激下加速逃离火源的行为。成年人、小孩和老人具有不同的加速上限，从而体现不同群体的运动能力差异。同时，当烟雾浓度较高时，人员速度会下降，以反映浓烟对视线、呼吸和行动能力的负面影响。该机制使火灾场景下的人群行为既包含恐慌逃离倾向，也包含受烟雾阻碍而行动受限的特征。

此外，系统采用局部重规划机制处理动态火灾对路径的影响。每次火灾状态更新后，系统会识别危险等级或阻断状态发生变化的 NavMesh 多边形，并查找当前路径经过这些区域的人员。只有当人员尚未走完的路径受到高代价或阻断区域影响时，才触发 Detour 重新规划。该方法避免了每一帧对所有人员进行全局重规划，在保证路径对火灾变化及时响应的同时，也降低了计算开销。

综上，火灾疏散场景具有明显的动态性和风险传播特征：火焰沿 NavMesh 邻接关系逐步蔓延，烟雾扩散持续改变空间风险分布，NavMesh 通行代价和可通行状态随时间更新，人员则根据火灾距离、烟雾浓度和自身类型调整路径与速度。通过上述机制，模型能够模拟火灾条件下人员绕避危险区域、动态重规划、靠近火源时加速逃离以及浓烟环境下行动受限等典型疏散行为。
