# 异质人群参数与家庭关系力模型

为提高人群疏散模拟的真实性，本文将行人划分为儿童、老人和成年人三类，并为不同类型个体设置差异化的身体参数、期望速度和灾害响应能力。设个体 $i$ 的类型为 $c_i$，其基础参数包括身高 $h_i$、个体半径 $r_i$ 和期望速度 $v_i^0$。

其中，个体半径用于描述行人在水平面上的占用空间，并参与人与人之间的硬碰撞分离、社会力排斥以及起点生成时的最小间距约束；期望速度表示个体在无拥堵、无障碍情况下的基础行走速度，是运动更新过程中的重要输入。

本文采用的三类人群参数如表 1 所示。

**表 1 不同类型行人的基础参数**

| 行人类型 | 生成比例 | 身高 / m | 半径 / m | 期望速度 / (m/s) |
|---|---:|---:|---:|---:|
| 儿童 | 20% | 1.20 | 0.20 | 0.90 |
| 老人 | 30% | 1.60 | 0.28 | 0.95 |
| 成年人 | 50% | 1.75 | 0.32 | 1.35 |

在火灾场景下，个体会根据其所处位置与火灾区域之间的 NavMesh 邻接距离产生速度响应。设火焰强度为 $I \in [0,1]$，火灾警戒半径定义为：

$$R_{\mathrm{fire}} = R_0 + R_I I$$

其中，$R_0 = 5\,\mathrm{m}$，$R_I = 7\,\mathrm{m}$。

当个体处于该警戒范围内时，其期望速度会乘以类型相关的加速倍率。成年人、儿童和老人的最大火灾速度倍率分别为 $1.50$、$1.40$ 和 $1.25$。为避免个体在警戒边界附近频繁切换状态，模型设置 $3\,\mathrm{m}$ 的迟滞区，即个体离开 $R_{\mathrm{fire}}+3\,\mathrm{m}$ 后才解除火灾加速状态。

同时，当烟雾浓度 $S$ 超过阈值 $S_{\mathrm{slow}} = 0.55$ 时，所有类型个体的速度倍率降为 $0.80$，用于表示浓烟对视线、呼吸和运动能力的抑制作用。

因此，个体 $i$ 的有效期望速度可表示为：

$$v_i^{\mathrm{eff}} = v_i^0 \lambda_i^{\mathrm{fire}} \lambda_i^{\mathrm{rain}}$$

其中，$\lambda_i^{\mathrm{fire}}$ 为火灾环境下的速度修正系数，$\lambda_i^{\mathrm{rain}}$ 为暴雨或积水等环境下的速度修正系数。若不处于对应灾害影响区域，则相应修正系数取 $1$。

除个体差异外，本文进一步考虑疏散过程中的家庭协同行为。真实疏散中，儿童往往倾向于靠近成年人，老人也会受到家庭成员照顾，因此个体并非完全独立运动。为刻画这一现象，本文引入视野约束的高斯家庭凝聚力。

设个体 $i$ 所属家庭为 $F(i)$，家庭成员 $j$ 与 $i$ 的距离为：

$$d_{ij} = \left\| \mathbf{x}_j - \mathbf{x}_i \right\|$$

方向单位向量为：

$$\mathbf{e}_{ij} = \frac{\mathbf{x}_j-\mathbf{x}_i}{\left\|\mathbf{x}_j-\mathbf{x}_i\right\|}$$

则个体 $i$ 受到的家庭凝聚力定义为：

$$\mathbf{F}_{\mathrm{fam}}(i) = \sum_{j \in F(i),\, j \ne i} V_{ij} w_{ij} g(d_{ij}) \mathbf{e}_{ij}$$

其中，$V_{ij}$ 为可见性约束项，$w_{ij}$ 为家庭成员类型关系权重，$g(d_{ij})$ 为距离相关的高斯凝聚函数。

可见性约束 $V_{ij}$ 定义如下：

$$V_{ij}=1,\quad \mathrm{if}\ d_{ij}<d_{\mathrm{view}}\ \mathrm{and}\ \mathrm{visible}(i,j)=1$$

$$V_{ij}=0,\quad \mathrm{otherwise}$$

其中，$d_{\mathrm{view}} = 30\,\mathrm{m}$。$\mathrm{visible}(i,j)=1$ 表示个体 $i$ 与个体 $j$ 之间在 NavMesh 上无遮挡；否则 $\mathrm{visible}(i,j)=0$。该约束避免了远距离或被墙体阻挡的家庭成员之间产生不合理的全局吸引。

距离相关函数 $g(d)$ 采用高斯钟形函数与近距离抑制项的乘积：

$$g(d)=\exp\left(-\frac{(d-d_{\mathrm{peak}})^2}{2\sigma^2}\right)\left[1-\exp\left(-\frac{d^2}{2d_{\min}^2}\right)\right],\quad 0 \le d < d_{\mathrm{view}}$$

当距离超过视野范围时：

$$g(d)=0,\quad d \ge d_{\mathrm{view}}$$

其中，$d_{\min}=1\,\mathrm{m}$ 表示近距离抑制范围，用于防止家庭成员过度聚集；$d_{\mathrm{peak}}=8\,\mathrm{m}$ 表示家庭凝聚力达到最大值的距离；$\sigma=6\,\mathrm{m}$ 控制高斯函数的衰减宽度；$d_{\mathrm{view}}=30\,\mathrm{m}$ 表示最大视野范围。

该函数使家庭凝聚力呈现“近距离弱、中距离强、远距离衰减、视野外消失”的特征。

家庭关系权重 $w_{ij}$ 根据两名家庭成员的类型确定，如表 2 所示。

**表 2 不同家庭成员类型之间的关系权重**

| 成员关系 | 权重 $w_{ij}$ |
|---|---:|
| 儿童--成年人 | 1.25 |
| 老人--成年人 | 0.90 |
| 老人--儿童 | 0.60 |
| 成年人--成年人 | 0.70 |
| 儿童--儿童 | 0.50 |
| 老人--老人 | 0.50 |

其中，儿童与成年人之间的权重最高，表示儿童在疏散过程中更依赖成年人；老人和成年人之间权重次之，用于刻画成年人对老人的照顾行为；同类成员之间权重相对较低，以避免家庭成员之间产生过强吸引导致局部拥堵。

在运动更新过程中，家庭凝聚力并不替代全局路径规划，而是作为局部运动修正项，与路径跟随力和人群排斥力共同作用。个体的期望运动方向可表示为：

$$\mathbf{d}_i=\operatorname{normalize}\left(\alpha\mathbf{d}_{\mathrm{path}}+\beta\mathbf{F}_{\mathrm{person}}+\gamma\mathbf{F}_{\mathrm{fam}}(i)\right)$$

其中，$\mathbf{d}_{\mathrm{path}}$ 为由 Detour 全局路径和路径带跟随机制给出的目标方向，$\mathbf{F}_{\mathrm{person}}$ 表示人与人之间的局部排斥力，$\mathbf{F}_{\mathrm{fam}}(i)$ 表示个体 $i$ 受到的家庭凝聚力，$\alpha$、$\beta$ 和 $\gamma$ 分别为三类行为项的权重。

最终期望速度为：

$$\mathbf{v}_i^{\mathrm{des}} = v_i^{\mathrm{eff}}\mathbf{d}_i$$

为防止个体间发生可见重叠，运动更新后还会进行硬碰撞分离。对于任意两个个体 $i$ 和 $j$，若二者距离满足：

$$d_{ij}<r_i+r_j+\epsilon$$

则沿二者连线方向进行位置修正，使最终距离不小于两者半径之和。其中，$\epsilon$ 为较小的安全间隙。该过程保证了即使在高密度或灾害加速情况下，个体之间仍不会出现明显重叠。

综上，本文通过异质人群参数、火灾速度响应、视野约束家庭凝聚力以及硬碰撞分离机制，将个体差异、社会关系和局部物理约束共同纳入疏散模拟过程，从而提高了人群行为的真实性和可解释性。
